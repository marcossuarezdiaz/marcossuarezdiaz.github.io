Tipos de selectores que entran en examen

Clase (class=””)

El atributo “class” se añade a una etiqueta y sirve para definir de que clase es esa etiqueta (es decir, a que clase corresponde).

Identificador (id=””)

ID es un atributo global y se aplica a prácticamente todos los elementos en HTML. Se utiliza para identificar elementos de forma exclusiva en la página web, y su valor se accede principalmente desde la interfaz.

Todos los elementos (*)

Al poner un (*) se aplica un estilo determinado a todos los elementos.

Varios elementos ( p, em, h1, h2)

El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos.

El elemento em sirve para poner el texto en cursiva.

La etiqueta h1 se utiliza para identificar el título principal del contenido de una página web.

La etiqueta h2 se utiliza para identificar un título secundario del contenido de una página web.

Elementos que van contenidos en otros elementos (ol li)

La etiqueta ul sirve para crear una lista ordenada, mientras que la etiqueta li sirve para poner los elementos en una lista, ya sea ordenada o desordenada.

Sobrevuleo (p:hover)

La rtiqueta p:hover se utiliza para que cuando el ratón esté encima de el texto que tiene asignada esta etiqueta cambie de color, al igual que un link pero sin necesidad de que lo sea.
